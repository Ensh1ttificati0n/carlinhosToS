Política de Privacidade

Carlinhos Bot · Última atualização: [DATA]

Esta política descreve que dados o Carlinhos Bot ("o Bot") recolhe, porquê, durante quanto tempo os guarda e como podes exercer os teus direitos. O Bot é operado por YanderuOverdose e serve o servidor de Discord The System Crash. Ao adicionar, utilizar ou permanecer num servidor onde o Bot está ativo, aceitas esta política.

Este documento reflete o comportamento real do Bot à data acima. Não vendemos, alugamos nem partilhamos os teus dados com terceiros para fins comerciais ou publicitários.

1. Que dados recolhemos

O Bot recolhe apenas os dados necessários ao seu funcionamento e às funcionalidades de moderação, registo e gestão do servidor:

  • Identificadores de Discord: ID de utilizador, ID de servidor, ID de canal, ID de mensagem, nome de utilizador, tag e apelido.
  • Conteúdo de mensagens: apenas quando relevante para funcionalidades de registo (por exemplo, o conteúdo de mensagens editadas ou apagadas é guardado para o sistema de logs).
  • Ficheiros e média: anexos de mensagens apagadas, e avatares/banners de membros, podem ser guardados em disco para o histórico de membros e comparação de imagens.
  • Registos de eventos (logs): entradas/saídas de membros, alterações de cargos e canais, ações de voz, e ações de moderação (bans, kicks, timeouts, avisos).
  • Ações de staff: registo das ações de moderação executadas pela equipa, para deteção de abuso e auditoria.
  • Dados de configuração: definições que os administradores configuram para o servidor.

O Bot não recolhe dados fora do Discord, não acede a mensagens privadas (DMs) entre utilizadores, e não recolhe informação de pagamento.

2. Porque recolhemos estes dados

  • Fornecer funcionalidades de moderação (bans, kicks, timeouts, avisos, anti-raid, anti-scam).
  • Manter registos (logs) de atividade do servidor para os administradores.
  • Operar sistemas de verificação, tickets e níveis/XP.
  • Detetar e prevenir abuso, incluindo abuso por parte da própria equipa de staff.
  • Guardar as preferências de configuração de cada servidor.

3. Base legal (RGPD)

Para utilizadores no Espaço Económico Europeu, o tratamento assenta no interesse legítimo em operar e moderar a comunidade, e no consentimento dado ao permaneceres num servidor onde o Bot está ativo e cujas regras indicam a sua utilização.

4. Durante quanto tempo guardamos os dados

  • Dados de configuração: enquanto o Bot estiver no servidor.
  • Logs de eventos e ações de staff: guardados em ficheiros locais. Alguns registos são mantidos sem prazo de eliminação automática, para fins de auditoria e histórico.
  • Média guardada (avatares, banners, anexos): mantida em disco até eliminação manual pelo operador.

Podes solicitar a eliminação dos teus dados a qualquer momento (ver secção 7).

5. Onde os dados são guardados

Os dados são guardados em: (a) uma base de dados MySQL alojada na Aiven (fornecedor cloud), e (b) ficheiros locais no servidor onde o Bot é executado. São aplicadas medidas técnicas razoáveis para proteger o acesso a estes dados.

6. Partilha com terceiros

Não vendemos nem partilhamos os teus dados para fins comerciais. Os dados são processados apenas pela infraestrutura necessária ao funcionamento do Bot (Discord, e o fornecedor de base de dados Aiven). Não existe partilha para publicidade.

7. Os teus direitos

Tens o direito de: aceder aos dados que temos sobre ti, corrigi-los, e solicitar a sua eliminação ("direito ao esquecimento"). Para exercer qualquer destes direitos, contacta-nos através dos meios na secção 9. Responderemos num prazo razoável.

8. Dados de menores

O Discord exige que os utilizadores tenham a idade mínima definida nos Termos de Serviço do Discord. O Bot não se destina a menores de idade abaixo dessa idade mínima e não recolhe intencionalmente os seus dados.

10. Alterações a esta política

Esta política pode ser atualizada. A data de "Última atualização" no topo reflete a versão em vigor. Alterações materiais serão comunicadas através do servidor.
